# Soporte de Arquitecturas

El framework RIF sigue creciendo mediante el sistema de plugins desacoplados. El objetivo es ampliar el core sólo cuando exista una necesidad común que no pueda resolverse limpiamente con un plugin, delegando las reglas específicas de arquitectura siempre al entorno de plugins.

## Arquitecturas Soportadas (Actualmente)

- **ARM7TDMI / Thumb** (Game Boy Advance)
- **Motorola 68000** (Sega Mega Drive / Genesis)
- **MOS 6502** (Atari 2600)
- **AMD64** (Soporte experimental inicial)

## Futuras Plataformas (Pendientes)

- **WDC 65C816** (Super Nintendo / SNES)
- **Zilog Z80** (Sega Master System / Game Boy Clásica)
- **MIPS R3000** (PlayStation 1)
