# Basics

`basics` es el plugin base.

Incluye instrucciones comunes:

- `need`
- `emit`
- `call`
- `exists`
- `fits`
- `eq` y `neq`
- `bitcat`
- `bitsize`
- `bitfit`
- `trunc`
- `zext`
- `sext`
- `lt`, `lte`, `gt`, `gte`
- `align`
- `pad`
- `reldis`
- `reloc`
- `emitadress`
- `error` y `raise`

No define una arquitectura. Solo aporta piezas genericas para construir reglas.
